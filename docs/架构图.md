## 二、模块一：知识库 RAG Pipeline（已实现 ✅）

```mermaid
flowchart LR
    subgraph 离线建库["📥 离线建库（文档→向量）"]
        A1["📄 原始文档<br/>PDF/Word/MD/TXT"] --> A2["🔧 Loader<br/>PyMuPDF · python-docx<br/>表格提取"]
        A2 --> A3["✂️ SemanticSplitter<br/>逐页语义切分<br/>sigma_mul 可配置"]
        A3 --> A4["🧮 Embedder<br/>BAAI/bge-base-zh-v1.5<br/>768维 本地免费"]
        A4 --> A5["🗄️ VectorStore<br/>ChromaDB HNSW<br/>持久化到磁盘"]
    end

    subgraph 在线检索["🔍 在线检索（问题→答案）"]
        B1["❓ 用户问题"] --> B2["🔬 QueryProcessor<br/>• 术语展开（138词）<br/>• LLM 扩写（≤15字触发）<br/>• 余弦校验（默认0.8）"]
        B2 --> B3["🏷️ EntityRouter<br/>公司实体识别<br/>单公司→过滤文档<br/>多公司→对比放行"]
        B3 --> B4["🔀 HybridSearch<br/>BM25(jieba) 关键词 ─┐<br/>语义向量检索 ────┤<br/>  ↓ RRF 融合(k=60)  │<br/>  ↓ 策略路由判断     │<br/>  ↓ Cross-Encoder重排 │"]
        B4 --> B5["📡 ModelRouter<br/>简单任务→flash(快+省)<br/>复杂推理→pro(深度推理)"]
        B5 --> B6["📝 答案生成<br/>LLM + Prompt约束<br/>引用溯源（文件+页码）"]
    end

    A5 --> B4
    B6 --> C["👤 用户看到<br/>答案 + 引用来源"]
```

## 三、模块二：数据分析 Agent（规划中 ⏳）

```mermaid
flowchart TB
    INPUT["👤 用户输入自然语言分析需求<br/>例如：对比茅台和比亚迪近三年毛利率趋势"]

    INPUT --> PLANNER["🧠 Planner 分析规划器<br/>• 分解为子任务列表<br/>• 确定需要的数据源和工具<br/>• 判断是否需要追问澄清"]

    PLANNER --> EXEC["⚡ Executor 执行器"]
    
    EXEC --> T1["🔍 工具：数据查询<br/>调用模块一 RAG<br/>调用模块三 MCP"]
    EXEC --> T2["🧮 工具：财务计算<br/>比率计算 · 杜邦分析<br/>现金流折现 · 趋势计算"]
    EXEC --> T3["📈 工具：图表生成<br/>matplotlib / ECharts<br/>折线·柱状·饼图·雷达图"]

    T1 --> MERGE["📊 结果汇总"]
    T2 --> MERGE
    T3 --> MERGE

    MERGE --> REPORTER["📝 Reporter 报告生成器<br/>• 结构化报告模板<br/>• 数据→图表→结论→建议"]
    
    REPORTER --> OUTPUT["📤 输出<br/>分析报告 · 可视化图表<br/>可导出 PDF/Excel"]
```

## 六、数据流全链路（端到端）

```mermaid
sequenceDiagram
    actor 用户
    participant 前端 as Streamlit 前端
    participant API as FastAPI
    participant RAG as 模块一 RAG
    participant Agent as 模块二 Agent ⏳
    participant MCP as 模块三 MCP ⏳
    participant LLM as DeepSeek API
    participant 外部 as Wind/同花顺

    用户->>前端: "茅台vs比亚迪近三年毛利率趋势，分析差异原因"
    前端->>API: POST /api/v1/analyze
    
    API->>Agent: 转发分析需求
    Agent->>Agent: Planner 拆解子任务
    
    par 并行数据获取
        Agent->>RAG: 子任务1: 检索茅台毛利率相关chunk
        RAG->>LLM: 语义检索
        RAG-->>Agent: 茅台毛利率数据
    and
        Agent->>RAG: 子任务2: 检索比亚迪毛利率相关chunk
        RAG-->>Agent: 比亚迪毛利率数据
    and
        Agent->>MCP: 子任务3: 获取行业平均毛利率
        MCP->>外部: 调用 Wind/同花顺
        外部-->>MCP: 行业数据
        MCP-->>Agent: 行业对标数据
    end
    
    Agent->>Agent: 计算趋势 + 对比差异
    Agent->>Agent: 生成图表 + 撰写结论
    Agent-->>API: 分析报告（数据+图表+结论）
    API-->>前端: JSON 响应
    前端-->>用户: 展示报告 + 交互图表
```
