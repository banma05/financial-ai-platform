## 一、项目概述

### 1.1 产品定位

**面向企业财务团队和金融分析师的 AI 智能分析平台**，覆盖"查数据 → 分析数据 → 写报告"全流程。

一句话：**自然语言驱动的财务数据分析助手。**

### 1.2 目标用户

| 用户角色 | 核心诉求 | 使用频率 |
|----------|----------|:----:|
| **金融分析师/投资经理** | 快速查阅年报、跨公司对比、生成分析报告 | 每日多次 |
| **企业内部财务人员** | 查询历史财务数据、生成内部汇报 | 每周 3-5 次 |
| **审计人员** | 核实数据来源、追溯原始披露 | 季度密集 |
| **非财务管理者** | 了解关键指标含义、快速获取经营数据 | 每月 2-3 次 |

### 1.3 核心价值

| 环节 | 人工方式 | AI 方式 | 提效 |
|------|----------|---------|:----:|
| 查数据 | 翻 PDF Ctrl+F → 人工判断上下文 | 自然语言提问 → 秒级定位 + 自动溯源 | **85%+** |
| 分析数据 | 手动提取 → Excel 计算 → 画图 | 自然语言驱动查询 → 自动计算 + 可视化 | **80%+** |
| 写报告 | 收集数据 → 整理格式 → 反复修改 | Agent 自动生成分析报告初稿 | **70%+** |

## 三、模块一：知识库（RAG 能力）✅ 已完成

### 3.1 业务场景

用户上传企业财报、公告、研报等文档后，通过自然语言提问，AI 自动检索文档内容并生成带引用的答案。

### 3.2 功能清单

| 功能 | 说明 | 状态 |
|------|------|:----:|
| 文档上传 | 支持 PDF / Word / Markdown / TXT，含表格提取 | ✅ |
| 智能问答 | 自然语言提问，AI 返回答案 + 原文引用（文件名、页码） | ✅ |
| 混合检索 | 语义搜索 + BM25(jieba中文分词+财务词典) → RRF融合 → Cross-Encoder重排 | ✅ |
| Query 处理 | 财务术语展开、Query 扩写、余弦相似度校验 | ✅ |
| 实体路由 | 公司实体识别 → 自动过滤无关文档，跨文档对比时放行 | ✅ |
| 模型路由 | 简单抽取→flash（省钱），复杂推理→pro（保效果） | ✅ |
| 评测体系 | 20题标准测试集 + R@5/MRR/NDCG + 数字归一化 + 按难度/类别统计 | ✅ |
| 参数实验 | chunk_size/overlap/语义阈值/Query阈值 4组对比实验 | ✅ |

### 3.3 数据源

| 类型 | 说明 | 当前状态 |
|------|------|:----:|
| 企业年报 | A 股/港股年度报告 PDF | 已接入（比亚迪/茅台/腾讯） |
| 企业公告 | 临时公告、重大事项披露 | 待扩充 |
| 券商研报 | 行业研究、公司深度报告 | 待扩充 |

### 3.4 评测基线

| 指标 | 单文档 | 多文档（3份年报） | 目标 |
|------|:------:|:-----------------:|:----:|
| **Recall@5** | 91.2% | **87.3%** | ≥90% |
| **MRR** | 85.8% | **90.8%** | ≥92% |
| NDCG@5 | — | 193.4% | ≥200% |

> 2026-06-30 基线。测试集 20 题，覆盖数值查询/趋势分析/对比分析/定义解释/风险分析/综合推理 6 类。

## 四、模块二：数据分析 Agent（Agent 能力）⏳ 规划中

### 4.1 业务场景

用户用自然语言提出分析需求（如"对比茅台和比亚迪近三年毛利率趋势，分析差异原因"），Agent 自主完成：查数据 → 计算指标 → 生成图表 → 撰写分析结论。

### 4.2 核心能力

| 能力 | 说明 | 优先级 |
|------|------|:------:|
| **NL 驱动数据查询** | 自然语言转结构化查询，自动从知识库/外部数据源提取数据 | P0 |
| **自动分析报告** | 根据分析主题，自动生成包含"数据→图表→结论→建议"的专业报告 | P0 |
| **可视化图表** | 自动生成折线图/柱状图/饼图/雷达图（财务指标对比、趋势、结构） | P0 |
| **多步推理** | 复杂分析问题拆解为多步骤（查→算→比→判），逐步执行 | P1 |
| **分析模板库** | 预设常见分析模板（杜邦分析、现金流分析、盈利能力评估） | P1 |
| **追问澄清** | 分析需求不明确时主动追问，确认范围和时间区间 | P2 |

### 4.3 典型使用场景

| 场景 | 用户输入 | Agent 行为 |
|------|----------|------------|
| 趋势分析 | "比亚迪近三年研发费用变化趋势，和营收增速对比" | 提取 3 年研发费用和营收 → 计算增速 → 双轴折线图 → 文字结论 |
| 对比分析 | "茅台和五粮液 2024 年盈利能力对比" | 提取毛利率/净利率/ROE → 分组柱状图 → 差异分析 |
| 风险扫描 | "帮我看看腾讯 2024 年有没有现金流风险" | 读取现金流表 → 计算经营/投资/筹资现金流比例 → 风险判断 |
| 自动报告 | "生成一份茅台 2024 年深度分析报告" | 执行多维度分析 → 汇总图表 → 生成结构化报告 |

## 五、模块三：MCP 工具集成 ⏳ 规划中

### 5.1 业务场景

为模块二的 Agent 提供"手和脚"——调用外部金融数据接口、执行财务计算，让 Agent 不仅能分析已上传的文档，还能获取实时/历史市场数据。

### 5.2 核心工具

| 工具 | 功能 | 优先级 |
|------|------|:------:|
| **Wind 数据接口** | 股票行情、财务数据、宏观指标（需 Wind 账号） | P0 |
| **同花顺数据接口** | 实时行情、历史数据、板块数据（免费/低成本） | P1 |
| **财务公式计算器** | 内置常用公式：杜邦分析、现金流折现、比率分析等 | P0 |
| **行业对标工具** | 自动获取同行业可比公司数据，生成对标分析 | P2 |
| **公告监控** | 监测目标公司最新公告，推送变动提醒 | P2 |

### 5.4 财务公式库（内置）

| 分类 | 公式示例 |
|------|----------|
| 盈利能力 | 毛利率、净利率、ROE、ROA、EBITDA 率 |
| 偿债能力 | 资产负债率、流动比率、速动比率、利息保障倍数 |
| 营运能力 | 存货周转率、应收账款周转率、总资产周转率 |
| 成长能力 | 营收增长率、净利润增长率、总资产增长率 |
| 估值指标 | PE、PB、PS、股息率、PEG |
| 现金流 | 自由现金流、经营现金流/净利润比率 |
| 杜邦分析 | ROE = 净利率 × 资产周转率 × 权益乘数 |
