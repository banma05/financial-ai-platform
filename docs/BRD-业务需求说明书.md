## 一、项目概述

### 1.1 产品定位

**面向企业财务团队和金融分析师的 AI 智能分析平台**，覆盖"查数据 → 分析数据 → 写报告"全流程。

一句话：**自然语言驱动的财务数据分析助手。**

### 1.2 目标用户

| 用户角色 | 核心诉求 | 使用频率 |
|----------|----------|:----:|
| **金融分析师/投资经理** | 快速查阅数据、跨公司对比、生成分析报告 | 每日多次 |
| **企业内部财务人员** | 查询历史财务数据、生成内部汇报 | 每周 3-5 次 |
| **审计人员** | 核实数据来源、追溯原始披露 | 季度密集 |
| **非财务管理者** | 了解关键指标含义、快速获取经营数据 | 每月 2-3 次 |

### 1.3 核心价值

| 环节 | 人工方式 | AI 方式 | 提效 |
|------|----------|---------|:----:|
| 查数据 | 翻 PDF Ctrl+F → 人工判断上下文 | 自然语言提问 → SQL毫秒级定位 | **95%+** |
| 分析数据 | 手动提取 → Excel 计算 → 画图 | 自动计算 + 可视化 | **80%+** |
| 写报告 | 收集数据 → 整理格式 → 反复修改 | Agent 自动生成分析报告 | **70%+** |

## 二、系统架构（V8.0 重构）

### 核心理念：SQL 优先查数字，RAG 辅助解读文本

```
用户输入："分析茅台 2024 年盈利能力"
         │
    ┌────▼────┐
    │ Planner │  LLM 拆解为子任务（0.5s）
    └────┬────┘
         │
    ┌────▼──────────────────────┐
    │        Executor            │
    │                            │
    │  ┌──────────┐ ┌──────────┐ │
    │  │ SQL 查询  │ │ RAG 辅助  │ │
    │  │ (毫秒级)  │ │ (文字解读) │ │
    │  └──────────┘ └──────────┘ │
    │  ┌──────────┐ ┌──────────┐ │
    │  │ 公式计算  │ │ 图表生成  │ │
    │  │ (Python)  │ │ (matplot)│ │
    │  └──────────┘ └──────────┘ │
    └────┬──────────────────────┘
         │
    ┌────▼────┐
    │ Reporter │  LLM 生成报告 + 图表（~3s）
    └─────────┘
         │
    ✅ 端到端 < 10 秒，数字 100% 准确
```

### 三大模块

| 模块 | 功能 | 数据来源 |
|------|------|----------|
| **SQL 结构化数据** | ROE/毛利率/营收等精确数字查询 | AKShare A股数据 + 规则提取 |
| **RAG 知识库** | 管理层讨论/风险提示/业务解读 | 年报文字章节 + 研报 |
| **Agent 分析引擎** | 任务拆解→执行→报告生成 | LLM 编排上述两模块 |

## 三、技术选型

| 组件 | 选型 | 说明 |
|------|------|------|
| LLM | DeepSeek v4-flash / v4-pro | flash=简单，pro=复杂推理 |
| Embedding | BAAI/bge-base-zh-v1.5 | 本地免费，768维 |
| Reranker | BAAI/bge-reranker-v2-m3 | 本地免费 |
| 向量库 | ChromaDB | 轻量免运维 |
| 数据源 | AKShare | A股免费金融数据 |
| 后端 | FastAPI + SSE | 流式响应 |
| 前端 | Streamlit | 快速原型 |
| Agent | LangGraph | 标准编排框架 |

## 四、验收指标（V8.0 目标）

| 指标 | 目标 | 评估方式 |
|------|:----:|----------|
| 数字查询准确率 | ≥99% | SQL 直接返回 vs 人工校对 |
| 端到端分析耗时 | ≤10 秒 | 从提问到报告生成 |
| 子任务拆解准确率 | ≥85% | 人工评估 Planner 输出 |
| 报告结构完整性 | ≥80% | 五章节完整性+引用质量 |
| 测试覆盖率 | ≥85% | CI 自动运行 |
